### Waht is Get Media Free?

This project, first called Free Video "Downloader" was started by a consistent
video donwloader necessitie for youtube video editor workflow and that kind 
of thing. 

Using yt-dlp that repository is an Graphical User Interface (GUI) implemented
with python and pyside6 to make a easy to use app. 

With this software you can download media content from different plataforms, 
here we use the yt-dlp by get the data from web, and was jus add a interface 
to more pratical use of the resources provides by getter core.

Get Media Free also uses ffmpeg to convert mp3 or mp4 if you want to get only
the audio from some media. And nodejs runtime to access youtube high quality 
data and more protected stuff like that.

All dependecies are embbed together with the .exe, so you dont need to have any
of this things installed on yout PC.
